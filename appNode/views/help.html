Aide

Pour le scan à un appareil

Il vous suffit de vous rendre sur la page correspondante et de chosir un mode de scan. Le premier permet de scanner avec votre appareil photo / caméra, et le second avec vos fichiers / photos présents sur votre appareil.

Pour le scan à deux appareils

Vous devez d'abord vous rendre sur la page "Réceptionner". Cette page vous propose un choix. Utiliser un code classique ou un qr code. Si vous choissisez le QR code, il vous faudra le scanner. Il vous enverra vers un nouvelle pas de scan. Après avoir reçu le texte contenu dans votre scan, il ne vous reste plus qu'a appuyer sur "démarrer le transfert", et votre texte devrait apparaître sur votre page de base "Réceptionner". Pour la solution du code classique, vous devez vous rendre manuellement sur la page "Scanner" sous "Scan à deux appareils". Là, comportez-vous comme un scan classique, lorsque vous recevez la réponse à votre scan, entrez le code affiché sur la page "Réceptionner". Puis, cliquez sur "démarrer le transfert". Le texte devrait apparaître sur l'autre page.
